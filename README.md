# APP 3 - PARCOURSUP 

# MEMBRES - 
Guillaume , Tom , Victor , Nathan 

# PRÉSENTATION DU PROGRAMME - 

Ce projet simule le mécanisme de ParcoursSup. Il charge un fichier CSV de candidatures, les trie par formation selon différents algorithmes (insertion, sélection, bulles, fusion, rapide, casier), puis génère pour chaque formation un classement avec admis et liste d'attente.
